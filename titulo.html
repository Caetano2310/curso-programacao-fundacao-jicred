  1. Primeiros passos na web

    como são sites construidos

    Todo site começa com HTML para estruturar o conteúdo, CSS para o visual e JavaScript para interatividade. Entender essa base ajuda você a ler qualquer página com mais clareza.

  2. navegadores e enereços

    URLs, abas favoritas

    O navegador transforma endereços digitado em páginas visíveis. Aprender a usar abas, favoritos e a barra de busca econmiza tempo no dia a dia.

  3. seguraça oline

    Senhas, links e privacidade

    Desconfie de links suspeitos, use senhas forte e verifique se o site usa conexão segura de informar dados pessoais.

    Proteja suas informações ao navegar

  4. Criando seu Primeiro Site

    Do arquivo local à publicação

    Salve seu HTML, abra no navegador para testar e publique em plataformas gratuitas para compartilhar seu trabalho com o mundo.
